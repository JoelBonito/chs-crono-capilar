## Insights para Design System

- **Cores Dominantes:** Branco Imaculado (#FFFFFF), Preto Profundo (#000000) e Ouro Premium (#ECB613).
- **Tipografia:** Uso de **Serifas Sofisticadas** (Newsreader/Cinzel style) para títulos e **Sans-serif limpo** (Manrope) para controles e corpo.
- **Geometria:** Bordas levemente arredondadas (Round 8) para botões e cards, transmitindo elegância e modernidade.
- **Vibe:** "French Pharmacy" meets High-End Beauty. Estética de luxo minimalista.
